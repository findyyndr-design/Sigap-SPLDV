Warung Kantin Sekolah
Misteri Es Teh & Gorengan
2 variabel · metode eliminasi · konteks kantin sekolah
🔓 Belum Terpecahkan
▼
📖 Cerita Kasus
Di kantin sekolah, Dinda membeli 2 gelas es teh dan 3 gorengan, ia membayar Rp7.000.

Sementara teman sebangkunya membeli 1 gelas es teh dan 5 gorengan, ia membayar Rp8.000.

Pak Kantin tidak memasang daftar harga. Bisakah kamu membantu mencari harga satuan?
🤔 Berapa harga 1 es teh dan 1 gorengan?
📐 Pemodelan SPLDV
x = harga 1 es teh
y = harga 1 gorengan
1
2x + 3y = 7.000
pembelian Dinda
2
x + 5y = 8.000
pembelian teman
🔢 Langkah Penyelesaian (Metode Eliminasi)
1
Samakan koefisien x — kalikan P2 dengan 2
P1: 2x + 3y = 7.000
P2 × 2: 2x + 10y = 16.000
2
Kurangkan P1 dari P2 × 2 → eliminasi x
(2x + 10y) − (2x + 3y) = 16.000 − 7.000
7y = 9.000 → y = 1.000
3
Substitusi y = 1.000 ke P1
2x + 3(1.000) = 7.000
2x = 4.000 → x = 2.000... tunggu!
3
Substitusi y = 1.000 ke P2
x + 5(1.000) = 8.000
x = 8.000 − 5.000 → x = 3.000
✅ Kasus Terpecahkan!
Harga Es Teh
Rp3.000
Harga Gorengan
Rp1.000
✔ Cek P1: 2(3.000) + 3(1.000) = 6.000 + 3.000 = 7.000 ✓
✔ Cek P2: 3.000 + 5(1.000) = 3.000 + 5.000 = 8.000 ✓
💡
Refleksi: Ternyata masalah sehari-hari di kantin sekolah bisa diselesaikan dengan SPLDV! Ketika ada dua hal yang tidak diketahui dan dua informasi berbeda, itulah saatnya SPLDV digunakan.

Transportasi Sehari-hari
Tarif Ojek & Angkot
2 variabel · metode substitusi · konteks transportasi
🔓 Belum Terpecahkan
▼
📖 Cerita Kasus
Andi berangkat sekolah naik ojek sejauh 2 km lalu lanjut 1 kali angkot. Total ongkosnya Rp18.000.

Rina yang tinggal lebih jauh naik ojek sejauh 1 km lalu 3 kali angkot. Ia membayar Rp21.000.

Pak ojek dan kondektur tidak memasang tarif resmi!
🤔 Berapa tarif ojek per km dan ongkos angkot per naik?
📐 Pemodelan SPLDV
x = tarif ojek per km
y = ongkos angkot per naik
1
2x + y = 18.000
perjalanan Andi
2
x + 3y = 21.000
perjalanan Rina
🔢 Langkah Penyelesaian (Metode Substitusi)
1
Nyatakan y dari Persamaan 1
2x + y = 18.000
y = 18.000 − 2x  … (3)
2
Substitusi (3) ke Persamaan 2
x + 3(18.000 − 2x) = 21.000
x + 54.000 − 6x = 21.000
−5x = −33.000 → x = 6.600
3
Substitusi x = 6.600 ke (3)
y = 18.000 − 2(6.600)
y = 18.000 − 13.200 → y = 4.800
✅ Kasus Terpecahkan!
Tarif Ojek/km
Rp6.600
Ongkos Angkot
Rp4.800
✔ Cek P1: 2(6.600) + 4.800 = 13.200 + 4.800 = 18.000 ✓
✔ Cek P2: 6.600 + 3(4.800) = 6.600 + 14.400 = 21.000 ✓
💡
Refleksi: Masalah tarif transportasi juga bisa diselesaikan pakai SPLDV! Metode Substitusi cocok dipakai saat salah satu persamaan mudah dinyatakan dalam variabel lain.

Warung Kelontong
Harga Mie & Telur
2 variabel · metode eliminasi · konteks warung kelontong
🔓 Belum Terpecahkan
▼
📖 Cerita Kasus
Bu Ani belanja di warung kelontong, membeli 3 bungkus mie dan 2 butir telur, membayar Rp13.000.

Kemudian Pak Budi datang membeli 5 bungkus mie dan 4 butir telur, membayar Rp24.000.

Label harga di warung sudah tidak terbaca!
🤔 Berapa harga 1 bungkus mie dan 1 butir telur?
📐 Pemodelan SPLDV
x = harga 1 bungkus mie
y = harga 1 butir telur
1
3x + 2y = 13.000
belanja Bu Ani
2
5x + 4y = 24.000
belanja Pak Budi
🔢 Langkah Penyelesaian (Metode Eliminasi)
1
Samakan koefisien y — kalikan P1 × 2
P1 × 2: 6x + 4y = 26.000
P2: 5x + 4y = 24.000
2
Kurangkan P2 dari P1 × 2 → eliminasi y
(6x + 4y) − (5x + 4y) = 26.000 − 24.000
x = 2.000 → x = 2.000
3
Substitusi x = 2.000 ke P1
3(2.000) + 2y = 13.000
6.000 + 2y = 13.000
2y = 7.000 → y = 3.500
✅ Kasus Terpecahkan!
Harga 1 Mie
Rp2.000
Harga 1 Telur
Rp3.500
✔ Cek P1: 3(2.000) + 2(3.500) = 6.000 + 7.000 = 13.000 ✓
✔ Cek P2: 5(2.000) + 4(3.500) = 10.000 + 14.000 = 24.000 ✓
💡
Refleksi: Hebat! Kamu sudah memecahkan 3 kasus! SPLDV ternyata sangat dekat dengan kehidupan kita sehari-hari. Sekarang saatnya masuk ke materi teori untuk memahami lebih dalam! 🚀
